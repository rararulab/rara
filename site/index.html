Not another chatbot.

Proactive heartbeat

A cron-driven background agent wakes up, reviews your recent sessions, and reaches out when it sees something worth mentioning. You don't have to ask.

Memory that never forgets

An append-only event tape records everything — messages, tool calls, decisions. Three layers of recall (working, episodic, semantic) survive restarts and scale without losing context.

Self-evolving soul

Rara's personality isn't static. It observes how you communicate, adapts its tone and interests over time, and stays within boundaries you define.

Security by architecture

A two-layer guard pipeline — taint tracking plus pattern scanning — runs before every tool call. Not a blacklist. A principled security model that scales.

Kernel, not middleware

Six modular components — LLM, tools, memory, sessions, guards, event bus — each independently replaceable. OS-inspired architecture, not a LangChain wrapper.

One agent, every channel

Telegram, web, CLI — same kernel, same memory, same personality. Deploy once, reach everywhere.

How it works.

01

Boot. The kernel loads your soul, mounts the tape, arms the guard pipeline, and starts the heartbeat.

02

Converse. Every message, tool call, and result is appended to an immutable tape. Context auto-folds when it hits 60% pressure.

03

Observe. Between conversations, the heartbeat reviews your sessions and decides whether to initiate contact.

04

Evolve. Over time, the soul adapts — new interests, adjusted tone, deeper understanding of your workflow.
