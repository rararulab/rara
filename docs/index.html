A self-evolving, proactive agent runtime built in Rust.
Kernel-inspired architecture. Heartbeat-driven. Multi-channel.
$ kernel — event-driven core with LLM, tools, memory, guard, and mood
$ proactive — heartbeat loop that acts without waiting for input
$ memory — 3-layer tape memory with session context and long-term recall
$ channels — web, telegram, wechat — one agent, many surfaces
$ skills — extensible capability system for tools and integrations
$ soul — personality and behavioral traits that shape every response
